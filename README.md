# jaejunchung.github.io

콘텐츠를 사랑하는 크리에이터 포트폴리오 웹사이트

## 🎬 소개

영화, 드라마, 책, 전시를 통해 일상의 영감을 나누는 개인 포트폴리오 웹사이트입니다.

## 📁 파일 구조

```
.
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 파일
└── README.md       # 프로젝트 문서
```

## ✨ 기능

- **반응형 디자인**: 모바일, 태블릿, 데스크톱에 최적화된 레이아웃
- **내비게이션**: 스무스 스크롤과 모바일 메뉴 토글
- **콘텐츠 필터링**: 영화, 드라마, 책, 전시 카테고리별 필터링
- **연락처 폼**: 메시지 전송 기능
- **인터랙티브 요소**: 호버 효과, 애니메이션, 스크롤 애니메이션

## 🎨 디자인 특징

- 모던하고 깔끔한 UI
- 그래디언트 배경과 카드 기반 레이아웃
- 부드러운 트랜지션과 애니메이션
- 접근성을 고려한 색상 대비

## 🚀 사용 방법

1. 파일을 웹 서버에 배포하거나 로컬에서 `index.html`을 브라우저로 엽니다
2. 네비게이션으로 다양한 섹션을 탐색합니다
3. 콘텐츠 필터를 사용하여 관심 있는 항목을 확인합니다
4. 연락처 섹션에서 메시지를 보냅니다

## 📝 커스터마이징

- `index.html`: 콘텐츠와 연락처 정보 수정
- `styles.css`: 색상 테마 및 스타일 변경
- `script.js`: 추가 인터랙티브 기능 구현
